Question-1: What are the different ways to select an element in the DOM?

YYou can select DOM elements using methods like getElementById (by ID), querySelector (first match), querySelectorAll (all matches), getElementsByTagName (by tag), and getElementsByClassName (by class). Each method serves different use cases based on the target.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets HTML content, including tags. innerText retrieves or sets visible text, respecting CSS styles like display: none. textContent gets or sets all text, ignoring tags and styles, and is faster than innerText.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is attached to a parent element, which handles events from child elements via event bubbling. This improves efficiency, especially for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM event mechanism where an event starts from the innermost element (target) and "bubbles up" to the outer parent elements, triggering their event listeners along the way. This allows multiple elements to respond to the same event.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements using JavaScript, first create an element with document.createElement("tagName"). To add the element, use appendChild() or insertBefore() to insert it into a parent element. To remove an element, use removeChild() to remove it from its parent. These methods allow you to dynamically manipulate the DOM.
